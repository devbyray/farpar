What is Farpar?

Farpar is a responsive HTML5 boilerplate that also reset the browser styles. Farpar is included with handy CSS presets, a basic stylesheet with the BEM syntax naming methodology and all the populair JavaScript libraries.

Use Farpar when you need a fundamental and don't want to start a new project from stratch.
