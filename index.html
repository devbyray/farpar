What is Farpar?

Farpar is an essential front-end boilerplate included with a responsive HTML5 template. It's based on a component/element principle and uses the following technologies:

  • npm
  • Grunt
  • Sass
  • Compass
  • AngularJS

Skipped columns

Skip an element within the grid for example with skip-1-6. No worrie about a direction with right to left. The skip function will respect your chosen direction.

Responsive columns

Make an element responsive for a small or large screen. Add for example the class small-width-1-4 to use that width on a small device. Or hide it on a large screen with the class large-hidden. Also skipping is possible on small or large screens. Add the class skip-large-1-3 for example.
