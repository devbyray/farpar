Farpar 2.0 - The ultimate front-end boilerplate

What is Farpar?

Farpar is a front-end boilerplate included with a responsive HTML5 template. It's based on a block/element principle and uses the following technologies:

  • npm
  • Grunt
  • Sass
  • Compass
  • AngularJS

We also introduce the Angle Class Syntax for CSS. See the section "Angle Class Syntax (ACS)" for examples.

Angle Class Syntax (ACS)

Ieder element krijgt standaard een class mee waarvan de naam gelijk is aan het element. Het voordeel is dat deze naam altijd kort en to the point is.

<p class="p">

Als er een p element is met een variant noem dan eerst de oorsprong en aansluitend tussen double angle quotes de variant. Als je gebruik maakt van Sass of Less extend dan de oorsprong in de variant class, dat scheelt een class naam in je HTML.

Voorbeeld zonder Sass of Less

<p class="p p«variant»">

Voorbeeld met Sass of Less (aanbevolen)

<p class="p«variant»">

Indien je niet de p class kunt hergebruiken, maak dan een nieuwe class aan ook beginnend met de naam van het element gevolgd door een underscore met daarachter je gewenste naam.

<p class="p_other»">

Anders dan bij de BEM syntax gebruik je bij elementen binnen een block niet dubbele underscores __ maar simpelweg een hyphen -. Dit is vooral overzichtelijker maar ook logisch. Je begint met het block en geeft na de hyphen aan wat daar binnen valt.

<div class="box">
  <p class="box-p"></p>
</div>
